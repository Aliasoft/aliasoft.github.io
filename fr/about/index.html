À propos

Plateformes logicielles, agents IA privés et architecture système.

Aliasoft, c’est plus de 10 ans d’expérience dans le développement de plateformes enterprise, de systèmes distribués et d’intégration de l’intelligence artificielle locale pour l’entreprise.

Portée par un ingénieur senior avec plus de 10 ans d'expérience en plateformes enterprise et systèmes distribués.

Aliasoft tire son nom de ses trois piliers : l'ingénieur (Al), l'intelligence artificielle (IA) et le logiciel (soft).

Domaines d’expertise

L’expertise d’Aliasoft couvre trois axes : l’ingénierie logicielle, le déploiement d’agents IA privés et l’architecture système. Priorité aux solutions souveraines, maintenables et pensées pour le long terme.

  • Systèmes backend Java / Spring Boot
  • Architecture event-driven avec Kafka
  • Interfaces React pour plateformes web modernes
  • Clean architecture et design inspiré du DDD
  • API REST de niveau production
  • Agents IA privés (Llama, Mistral) - on-premise
  • Bases de connaissance métier (RAG)
  • APIs d’accès aux données pour LLM
  • Déploiements Docker, Nginx, CI/CD

Focus actuel - Aliasoft

Aliasoft a livré un écosystème maritime complet (Seaway, SeaScanner, SeaBook), assuré des formations techniques en école d'ingénieurs et développe un axe agents IA privés pour l'entreprise (Llama, Mistral, RAG). Disponible pour de nouvelles missions. Stack : Java 21, Spring Boot 3, Kafka, React, Docker, Ollama.
